## Patch instructions  🩹

1. Visit [Rom Patcher JS](https://www.marcrobledo.com/RomPatcher.js/)
2. Download the latest xdelta patch in this folder, and use it as the Patch file.
   - Note: This is a **rolling release**: check frequently for the latest fixes/improvements.
3. Select `Densetsu no Stafy 4 (Japan).nds` as ROM file
   - Correct crc32: [45fe3a37](https://datomatic.no-intro.org/index.php?page=show_record&s=28&n=0401))
5. Click "Apply patch".
